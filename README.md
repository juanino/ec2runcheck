# ec2runcheck
Check if csv with ec2 instances is running

# get started

* works with a csv of 10 columns
* git clone git@github.com:juanino/ec2runcheck.git
* cd ec2runcheck
* python3 -m venv .
* source bin/activate
* pip install -r requirements.txt
* change somefile.csv in tag.py to your file
* ./tag.py
